#  Spotify SQL Analytics

Pet-project для демонстрации навыков работы с SQL и аналитикой данных на реальных данных Spotify.

В проекте используются открытые датасеты Spotify (2009–2025), содержащие информацию о треках, артистах, альбомах и их характеристиках.
Основной фокус — загрузка «сырых» CSV-данных в PostgreSQL, приведение их к аналитически удобному виду и выполнение запросов для исследования музыкальных трендов.

Проект показывает работу с:
	•	импортом данных из CSV в PostgreSQL
	•	типами данных и их нормализацией
	•	аналитическими SQL-запросами
	•	обработкой «грязных» данных (даты, жанры, числовые признаки)

На основе подобного пайплайна можно решать реальные задачи музыкального стриминга и контент-платформ:

	•	Анализ популярности треков и артистов
	•	Анализ музыкальных трендов
	•	Поддержка рекомендационных систем
	•	Контент-стратегия и плейлисты
	•	Поиск закономерностей между характеристиками треков и их популярностью для формирования плейлистов и рекомендаций.
	•	Подготовка SQL-витрин данных для дашбордов и аналитических отчётов.
